# [71. 简化路径](https://leetcode.cn/problems/simplify-path/)



用栈来考虑，除开‘/’其他所有都视作一个字符串，然后保存到栈中，然后下一步从栈中取出字符串按步骤考虑是否舍弃或存入结果。

```java
class Solution {
    public String simplifyPath(String path) {
        Stack<String> pathStack = new Stack<>();
        int begin;
        begin = 0;
        boolean slash = true;
        for(int i = 0; i < path.length(); i ++){
            char c = path.charAt(i);
            if(c == '/'){
                if(slash == false){
                    pathStack.push(path.substring(begin+1,i));
                }
                slash = true;
                begin = i;
            }
            else{
                slash = false;
            }
        }
        if(slash != true){
            pathStack.push(path.substring(begin+1));
        }
        Stack<String> resStack = new Stack<>();
        int jump = 0;
        while(!pathStack.empty()){
            String s = pathStack.pop();
            if(s.equals(new String("."))){
                continue;
            }
            else if(s.equals(new String(".."))){
                jump ++;
            }
            else{
                if(jump > 0){
                    jump --;
                }
                else{
                    resStack.push(s);
                }
            }
        }
        StringBuilder res = new StringBuilder();
        if(resStack.empty()){
            res.append("/");
        }
        while(!resStack.empty()){
            res.append("/");
            res.append(resStack.pop());
            
        }
        return res.toString();
    }
}
```
