# [1249. 移除无效的括号](https://leetcode.cn/problems/minimum-remove-to-make-valid-parentheses/)

栈放左括号就行

```java
class Solution {
    public String minRemoveToMakeValid(String s) {
        char left = '(';
        char right = ')';
        Stack<Character> st = new Stack<>();
        StringBuilder res = new StringBuilder();
        for(char c : s.toCharArray()){
            if(c == left || c == right){
                if(c == left){
                    st.push(c);
                    res.append(c);
                }
                else{
                    if(!st.empty()){
                        st.pop();
                        res.append(c);
                    }
                }
            }
            else{
                res.append(c);
            }
        }
        if(st.empty())
            return res.toString();
        else{
            int i = res.length() - 1;
            while(i >= 0){
                if(res.charAt(i) == left){
                    if(!st.empty()){
                        st.pop();
                        res.deleteCharAt(i);
                    }
                }
                i --;
            }
            return res.toString();
        }
    }
}
```
