# 就业准备笔记仓库

本仓库用于系统化记录就业准备相关的知识与实践，刷题只是其中一部分，后续可加入算法、计算机基础、工程实战、八股与面经等内容。

## 仓库结构

- `LeetCode专题 - 328题`：分专题整理的刷题题解与思路
- `代码随想录 - 185题`：按训练路径整理的经典题目
- `LeetCode 每日一题`：每日练习记录与解析
- 更多主题将逐步补充：`计算机基础/`、`系统设计/`、`后端工程/`、`面试准备/` 等

## 快速导航

- 刷题笔记索引 → `./LeetCode-Index.md`
- 推荐笔记模板 → `./NOTE_TEMPLATE.md`

## 使用建议

- 新增笔记建议统一采用模板，便于索引与检索
- 按主题归档到对应目录，保持文件名含题号或关键词，便于定位
- 每个主题下维护一个 `README` 或索引页面，聚合跳转与常用参考
