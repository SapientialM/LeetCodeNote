# [202. 快乐数](https://leetcode.cn/problems/happy-number/)

```java
class Solution {
    public boolean isHappy(int n) {
        // 用Set判断是否出现了循环
        HashSet<Integer> hashSet = new HashSet<>();
        hashSet.add(n);
        int res = n;
        int num;
        while(res != 1){
            num = res;
            int offset = 1;
            while(res >= 10){
                res = res / 10;
                offset ++;
            }
            res = 0;
            for(int i = 0; i < offset; i ++){
                int p = num % 10;
                num /= 10;
                res += p * p;
            }
            if(hashSet.contains(res)){
                return false;
            }
            hashSet.add(res);
        }
        return true;

    }
}
```
