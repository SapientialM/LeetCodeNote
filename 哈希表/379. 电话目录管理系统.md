# [379. 电话目录管理系统](https://leetcode.cn/problems/design-phone-directory/)

```java
class PhoneDirectory {
    // k-v
    ArrayList<Boolean> phoneMenu;
    public PhoneDirectory(int maxNumbers) {
        phoneMenu = new ArrayList<>(maxNumbers);
        for(int i = 0; i < maxNumbers; i ++){
            phoneMenu.add(true);
        }
    }
    
    public int get() {
        for(int i = 0; i < phoneMenu.size(); i++){
            if(phoneMenu.get(i) == true){
                phoneMenu.set(i, false);
                return i;
            }
        }
        return -1;
    }
    
    public boolean check(int number) {
        return phoneMenu.get(number);
    }
    
    public void release(int number) {
        phoneMenu.set(number, true);
    }
}

 * Your PhoneDirectory object will be instantiated and called as such:
 * PhoneDirectory obj = new PhoneDirectory(maxNumbers);
 * int param_1 = obj.get();
 * boolean param_2 = obj.check(number);
 * obj.release(number);
 */
```
