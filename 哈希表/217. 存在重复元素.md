# [217. 存在重复元素](https://leetcode.cn/problems/contains-duplicate/)

```java
class Solution {
    public boolean containsDuplicate(int[] nums) {
        HashSet<Integer> hashSet = new HashSet<>();
        for(int i : nums){
            if(hashSet.contains(i)){
                return true;
            }
            hashSet.add(i);
        }
        return false;
    }
}
```
