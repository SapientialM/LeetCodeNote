# [121. 买卖股票的最佳时机](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/)

```java
class Solution {
    public int maxProfit(int[] prices) {
        int minPrice = 10001;
        int profit = 0;
        for(int i : prices){
            profit = Math.max(i - minPrice, profit);
            if(i < minPrice) minPrice = i;
        }
        return profit;
    }
}
```
