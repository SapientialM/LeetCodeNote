# [123. 买卖股票的最佳时机 III](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iii/)

双线操作，难度立马就上来了。

我们需要对交易进行定义，一次买入，一次卖出则为一笔交易，我们最多可以完成两笔交易。

## 状态转移

我们**一共最多完成两笔交易**，那么一天一共就有五个操作状态，

> 这个状态解释一下，以第0天为例子，一天可以交易两次，那么就会有几个状态，不买，买，卖买，买买，买卖

0. **没有操作**（不买不卖）
1. **第一次买入后**（持有第一支股票）
2. **第一次卖出后**（完成第一次交易）
3. **第二次买入后**（持有第二支股票）
4. **第二次卖出后**（完成第二次交易）



## 递推公式

**以第0天为例**（假设股价为3）：

状态0：没有操作 → 利润 = 0
状态1：第一次买入 → 利润 = -3（花了3元买股票）
状态2：第一次卖出 → 利润 = 0（买入又卖出，不赚不赔）
状态3：第二次买入 → 利润 = -3（完成一次交易后再买入）
状态4：第二次卖出 → 利润 = 0（完成两次交易）

**以第1天为例**（假设第0天股价为3，第1天股价为5）：

状态0：没有操作 → 利润 = 0
状态1：第一次买入 → 利润 = -3（保持第0天的低价买入，比今天买入更划算）
状态2：第一次卖出 → 利润 = 2（卖出第0天买入的股票，赚2元）
状态3：第二次买入 → 利润 = -3（用第一次交易的利润买入第二支股票）
状态4：第二次卖出 → 利润 = 2（卖出第二支股票，完成两次交易）

```java
class Solution {
    public int maxProfit(int[] prices) {
        int n = prices.length;
        int dp[][] = new int[n][5];
        dp[0][0] = 0;
        dp[0][1] = -prices[0]; dp[0][2] = 0;
        dp[0][3] = -prices[0]; dp[0][4] = 0;
        for(int i = 1; i < n; i ++){
            dp[i][0] = dp[i-1][0];
            dp[i][1] = Math.max(dp[i-1][0] - prices[i], dp[i-1][1]);
            dp[i][2] = Math.max(dp[i-1][1] + prices[i], dp[i-1][2]);
            dp[i][3] = Math.max(dp[i-1][2] - prices[i], dp[i-1][3]);
            dp[i][4] = Math.max(dp[i-1][3] + prices[i], dp[i-1][4]);
        }
        return dp[n-1][4];
    }
}
```
