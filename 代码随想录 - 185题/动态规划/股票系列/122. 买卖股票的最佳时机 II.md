# [122. 买卖股票的最佳时机 II](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-ii/)

之前用贪心做的，把每天买卖都合在一起，收益大于0就作为收益。

现在我们来做动态规划：

```java
class Solution {
    public int maxProfit(int[] prices) {
        int n = prices.length;
        int dp[][] = new int[n][2];
        // dp[i][] 表示第i天的股票持有情况
        // 0 是今天没有持有股票，1是持有股票,只记录收益
        dp[0][0] = 0; dp[0][1] = -prices[0];
        for(int i = 1; i < n; i ++){
            // 如果今天没持有，就两种，昨天没持有，昨天持有今天卖了
            dp[i][0] = Math.max(dp[i-1][0], dp[i-1][1] + prices[i]);
            // 如果今天持有，就两种，昨天持有今天保持，昨天没有今天买的
            dp[i][1] = Math.max(dp[i-1][1], dp[i-1][0] - prices[i]);            
        }
        // 最后一天一定是要不持有才是最大收益。
        return dp[n-1][0];
    }
}
```
