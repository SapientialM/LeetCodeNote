# [53. 最大子数组和](https://leetcode.cn/problems/maximum-subarray/)

用了子序列dp后感觉舒服多了。

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int n = nums.length;
        int dp[] = new int[n];
        Arrays.fill(dp, Integer.MIN_VALUE);
        int res = nums[0];
        dp[0] = nums[0];
        for(int i = 1; i < n; i ++){
            dp[i] = Math.max(nums[i], dp[i-1] + nums[i]);
            res = Math.max(res, dp[i]);
        }
        return res;
    }
}
```
