# [647. 回文子串](https://leetcode.cn/problems/palindromic-substrings/)

```java
class Solution {
    public int countSubstrings(String s) {
        // 二维dp
        int n = s.length();
        // dp[i][j] 意味着 i到j是 回文串
        boolean dp[][] = new boolean[n][n];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                dp[i][j] = false;  // 其实不需要，默认就是 false
            }
        }
        int res = 0;
        for(int i = n-1; i >= 0; i --){
            for(int j = i; j < n; j ++){
                if(s.charAt(i) == s.charAt(j)){
                    if(j-i < 2){
                        // 短串一定是回文
                        dp[i][j] = true;
                        res ++;
                    }
                    else{
                        // 看子串是否回文
                        if(dp[i+1][j-1]){
                            dp[i][j] = true;
                            res ++;
                        }
                    }
                }
            }
        }
        return res;
    }
}
```
