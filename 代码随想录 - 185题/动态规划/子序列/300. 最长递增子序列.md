# [300. 最长递增子序列](https://leetcode.cn/problems/longest-increasing-subsequence/)

找到状态表达式和转移公式即可

```java
class Solution {
    public int lengthOfLIS(int[] nums) {
        // dp五部曲
        int n = nums.length;
        int dp[] = new int[n];
        int res = 0;
        dp[0] = 1;
        // dp[i] 表示以 nums[i]结尾的最长递增子序列长度
        for(int i = 0; i < n; i ++){
            dp[i] = 1;
            for(int j = 0; j < i; j ++){
                if(nums[i] > nums[j]) dp[i] = Math.max(dp[j] + 1, dp[i]);
            }
            if(dp[i] > res) res = dp[i];
        }
        return res;
    }
}
```
