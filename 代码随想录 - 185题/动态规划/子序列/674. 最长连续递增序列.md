# [674. 最长连续递增序列](https://leetcode.cn/problems/longest-continuous-increasing-subsequence/)

```java
class Solution {
    public int findLengthOfLCIS(int[] nums) {
        // 连续
        int n = nums.length;
        if(n < 2) return 1;
        int dp[] = new int[n];
        Arrays.fill(dp, 1);
        int res = 0;
        for(int i = 1; i < n; i ++){
            if(nums[i] > nums[i-1]) dp[i] = Math.max(dp[i], dp[i-1] + 1);
            res = Math.max(res, dp[i]);
        }
        return res;
    }
}
```
