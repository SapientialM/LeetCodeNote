# [63. 不同路径 II](https://leetcode.cn/problems/unique-paths-ii/)

多了个障碍物，思路差不多

```java
class Solution {
    public int uniquePathsWithObstacles(int[][] obstacleGrid) {
        int m = obstacleGrid.length;
        int n = obstacleGrid[0].length;
        
        // 如果起点或终点是障碍物，直接返回0
        if (obstacleGrid[0][0] == 1 || obstacleGrid[m-1][n-1] == 1) {
            return 0;
        }
        
        int[][] dp = new int[m][n];
        
        // 初始化起点
        dp[0][0] = 1;
        
        // 初始化第一列
        for (int i = 1; i < m; i++) {
            if (obstacleGrid[i][0] == 1) {
                dp[i][0] = 0; // 遇到障碍物，后面的都无法到达
            } else {
                dp[i][0] = dp[i-1][0]; // 只能从上方来
            }
        }
        
        // 初始化第一行
        for (int j = 1; j < n; j++) {
            if (obstacleGrid[0][j] == 1) {
                dp[0][j] = 0; // 遇到障碍物，后面的都无法到达
            } else {
                dp[0][j] = dp[0][j-1]; // 只能从左方来
            }
        }
        
        // 填充其余位置
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (obstacleGrid[i][j] == 1) {
                    dp[i][j] = 0; // 障碍物位置无法到达
                } else {
                    dp[i][j] = dp[i-1][j] + dp[i][j-1]; // 从上方或左方来
                }
            }
        }
        
        return dp[m-1][n-1];
    }
}
```

空间优化：
```java
class Solution {
    public int uniquePathsWithObstacles(int[][] obstacleGrid) {
        int m = obstacleGrid.length;
        int n = obstacleGrid[0].length;
        
        if (obstacleGrid[0][0] == 1 || obstacleGrid[m-1][n-1] == 1) {
            return 0;
        }
        
        int[] dp = new int[n];
        dp[0] = 1;
        
        for (int j = 1; j < n; j++) {
            dp[j] = (obstacleGrid[0][j] == 1) ? 0 : dp[j-1];
        }
        
        for (int i = 1; i < m; i++) {
            // 更新第一列
            dp[0] = (obstacleGrid[i][0] == 1) ? 0 : dp[0];
            
            for (int j = 1; j < n; j++) {
                if (obstacleGrid[i][j] == 1) {
                    dp[j] = 0;
                } else {
                    dp[j] = dp[j] + dp[j-1];
                }
            }
        }
        
        return dp[n-1];
    }
}
```
