# [70. 爬楼梯](https://leetcode.cn/problems/climbing-stairs/)

很简单，反向递归就行，但是递归会超时，毕竟复杂度太高了

```java
class Solution {
    public int climbStairs(int n) {
        if(n == 0) return 1;
        if(n < 0) return 0;

        return climbStairs(n-1) + climbStairs(n-2);
    }
}
```

所以做优化，除开递归之外，动态规划经典解法，DP数组，从0和1阶开始推理

```java
class Solution {
    public int climbStairs(int n) {
        if (n <= 1) return 1;
        
        int[] dp = new int[n + 1];
        dp[0] = 1; // 到达0阶台阶有1种方式（不爬）
        dp[1] = 1; // 到达1阶台阶有1种方式（爬1步）
        
        for (int i = 2; i <= n; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        
        return dp[n];
    }
}

```
