# [343. 整数拆分](https://leetcode.cn/problems/integer-break/)

**思路**：

1. 从小的数字开始算起，比如先算2的最大乘积是1（只能拆成1+1）
2. 算3的时候，尝试所有拆分方式：1+2、1+1+1，取乘积最大的
3. 算4的时候，可以拆成1+3、2+2、1+1+2等，取乘积最大的
4. 以此类推，每个大数字的最大乘积都依赖于前面小数字的计算结果

**关键技巧**：
- 对于数字i，我们尝试拆成j和(i-j)
- 然后比较两种选择：要么直接相乘j×(i-j)，要么把(i-j)继续拆分后的最大乘积再乘以j
  - i、j 拆分是可以遍历完的
  - 而 dp[i-j] 是会包含非两个数相乘的情况的

- 取这两种情况中更大的那个

简单说就是：**把大问题分解成小问题，利用小问题的答案来解大问题**。

```java
class Solution {
    public int integerBreak(int n) {
        int[] dp = new int[n+1];
        dp[2] = 1;
        for(int i = 3; i <= n; i ++){
            for(int j = 1; j <= i; j ++){
                dp[i] = Math.max(dp[i], Math.max((i-j)*j, dp[i-j]*j));
            }
        }
        return dp[n];
    }
}
```
