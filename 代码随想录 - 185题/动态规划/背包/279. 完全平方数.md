# [279. 完全平方数](https://leetcode.cn/problems/perfect-squares/)

表面是平方数，实际上只要遍历 1~sqrt(n)，每次取  i*i ，你就可以得到平方数，所以还是背包

```java
class Solution {
    // 遍历物品, 再遍历背包
    public int numSquares(int n) {
        int[] dp = new int[n + 1];
        //初始化
	    Arrays.fill(dp, Integer.MAX_VALUE);
	
        //当和为0时，组合的个数为0
        dp[0] = 0;
        // 遍历物品（从1~n的所有平方数），还是同样的，要不要取物品 i*i
        for (int i = 1; i * i <= n; i++) {
            // 遍历背包，还是同样的 j 表示容量，每次权衡大小，大小更新是取一个数就 + 1
            for (int j = i * i; j <= n; j++) {
                // 这里的 j 就一定会是平方数
                dp[j] = Math.min(dp[j], dp[j - i * i] + 1);
            }
        }
        return dp[n];
    }
}
```
