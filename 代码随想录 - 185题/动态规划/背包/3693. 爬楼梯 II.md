# [3693. 爬楼梯 II](https://leetcode.cn/problems/climbing-stairs-ii/)

从每个台阶起步，向后维护可达的台阶最小成本：
```java
class Solution {
    public int climbStairs(int n, int[] costs) {
        // dp[i] 表示到达台阶 i 的最小总成本
        int[] dp = new int[n + 1];
        Arrays.fill(dp, Integer.MAX_VALUE);
        dp[0] = 0;
        
        for (int i = 0; i < n; i++) {
            // 如果当前台阶不可达，跳过
            if (dp[i] == Integer.MAX_VALUE) continue;
            
            // 尝试跳到 i+1, i+2, i+3
            for (int j = i + 1; j <= Math.min(n, i + 3); j++) {
                // costs 数组下标从 1 开始，所以 costs[j] 对应 costs[j-1]
                int costJump = costs[j - 1] + (j - i) * (j - i);
                if (dp[i] + costJump < dp[j]) {
                    dp[j] = dp[i] + costJump;
                }
            }
        }
        
        return dp[n];
    }
}
```
