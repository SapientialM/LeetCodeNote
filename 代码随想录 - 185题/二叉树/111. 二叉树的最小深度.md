# [111. 二叉树的最小深度](https://leetcode.cn/problems/minimum-depth-of-binary-tree/)

和最大深度不一样，最小深度多了一个判断，需要判断是否为底。先从最简单的递归开始。

```java
class Solution {
    public int minDepth(TreeNode root) {
        if(root == null){
            return 0;
        }
        int leftDepth = Integer.MAX_VALUE;
        int rightDepth = Integer.MAX_VALUE;
        if(root.left != null){
            leftDepth = minDepth(root.left);
        }
        if(root.right != null){
            rightDepth = minDepth(root.right);
        }
        if(root.left == null && root.right == null){
            // 到底了，到底部一定是到这里，所以不会出现问题
            leftDepth = 0;
            rightDepth = 0;
        }
        return Math.min(leftDepth, rightDepth) + 1;
    }
}
```

那么现在来看看如何改造递归，也是判断到底的问题。

```java
class Solution {
    public int minDepth(TreeNode root) {
        if (root == null) return 0;
        
        Stack<Pair<TreeNode, Integer>> stack = new Stack<>();
        stack.push(new Pair<>(root, 1));
        int minDepth = Integer.MAX_VALUE;
        // DFS，深度遍历，要用栈，且因为需要记录depth信息，所以还需要使用depth。
        // 用 Pair 可以简化代码
        while (!stack.isEmpty()) {
            Pair<TreeNode, Integer> pair = stack.pop();
            TreeNode node = pair.getKey();
            int currentDepth = pair.getValue();
            if(node.left == null && node.right == null){
                // 逻辑和递归是一样的，到底了才开始维护高度
                minDepth = Math.min(minDepth, currentDepth);
            }
            
            if (node.right != null) {
                stack.push(new Pair<>(node.right, currentDepth + 1));
            }
            if (node.left != null) {
                stack.push(new Pair<>(node.left, currentDepth + 1));
            }
        }
        
        return minDepth;
    }
}
```
