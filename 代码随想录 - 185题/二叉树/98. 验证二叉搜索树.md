# [98. 验证二叉搜索树](https://leetcode.cn/problems/validate-binary-search-tree/)

找到BST的顺序，即可验证是否为BST。

```java
class Solution {
    public boolean isValidBST(TreeNode root) {
        List<Integer> dataList = new ArrayList<>();
        traversal(root, dataList);
        int pre = dataList.get(0);
        for(int i = 1; i < dataList.size(); i ++){
            int val = dataList.get(i);
            if(pre >= val){
                return false;
            }
            pre = val;
        }
        return true;
    }
    public void traversal(TreeNode node, List<Integer> dataList){
        if(node == null){
            return;
        }
        // 遍历到最左，一定是最小的
        traversal(node.left, dataList);
        dataList.add(node.val);
        traversal(node.right, dataList);
    }
}
```
