# [654. 最大二叉树](https://leetcode.cn/problems/maximum-binary-tree/)

其实这种题目，有时候我真的不是很理解。它没有很明显的说明白你如何去构建这个树，这道题其实是给你中序遍历的结果，然后构建的。

1. 找到最大值，创建节点
2. 从节点切分左右
3. 递归

```java
class Solution {
    public int findMaxWithIndex(int[] nums){
        int index = 0;
        int maxValue = Integer.MIN_VALUE;
        for(int i = 0; i < nums.length; i ++){
            if(nums[i] > maxValue){
                index = i;
                maxValue = nums[i];
            }
        }
        return index;
    }

    public TreeNode constructMaximumBinaryTree(int[] nums) {
        if(nums.length == 0) return null;
        if(nums.length == 1) return new TreeNode(nums[0]);
        int max = findMaxWithIndex(nums);
        TreeNode root = new TreeNode(nums[max]);
        root.left = constructMaximumBinaryTree(Arrays.copyOfRange(nums, 0, max));
        root.right = constructMaximumBinaryTree(Arrays.copyOfRange(nums, max+1, nums.length));
        return root;
    }
}
```
