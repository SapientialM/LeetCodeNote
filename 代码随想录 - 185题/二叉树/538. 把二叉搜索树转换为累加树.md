# [538. 把二叉搜索树转换为累加树](https://leetcode.cn/problems/convert-bst-to-greater-tree/)

```java
class Solution {
    public TreeNode convertBST(TreeNode root) {
        calSum(root, 0);
        return root;
    }
    private int calSum(TreeNode root, int sum){
        if(root == null) return sum;
        // 最右侧是最终数，也是起始
        if(root.left == null && root.right == null){
            root.val = root.val + sum;
            return root.val;
        }
        // 向右传sum，因为无贡献
        root.val += calSum(root.right, sum);
        // 向左传自己，因为有贡献
        return calSum(root.left, root.val);
    }
}
```
