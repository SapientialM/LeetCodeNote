# [226. 翻转二叉树](https://leetcode.cn/problems/invert-binary-tree/)

层序遍历+翻转子节点

```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        if(root == null) return null;
        // 层序遍历，在遍历基础上加入交换节点
        Queue<TreeNode> nodeQ = new ArrayDeque<>();
        nodeQ.offer(root);
        while (!nodeQ.isEmpty()) {
            int size = nodeQ.size();
            for(int i = 0; i < size; i ++){
                TreeNode node = nodeQ.poll();
                TreeNode left = null;
                TreeNode right = null;
                if(node.right != null) {
                    nodeQ.offer(node.right);
                    right = node.right;
                }
                if(node.left != null){
                    nodeQ.offer(node.left);
                    left = node.left;
                }
                node.left = right;
                node.right = left;
            }
        }
        return root;
    }
}
```
