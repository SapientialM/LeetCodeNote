# [700. 二叉搜索树中的搜索](https://leetcode.cn/problems/search-in-a-binary-search-tree/)

```java
class Solution {
    public TreeNode searchBST(TreeNode root, int val) {
        TreeNode findNode = root;
        while (findNode != null) {
            if(findNode.val == val){
                return findNode;
            }
            if(findNode.val < val){
                findNode = findNode.right;
            }
            else{
                findNode = findNode.left;
            }
        }
        return null;
    }
}
```
