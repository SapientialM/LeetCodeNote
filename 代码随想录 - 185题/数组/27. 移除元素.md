# [27. 移除元素](https://leetcode.cn/problems/remove-element/)

双指针解决，一个探测，一个填充

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        // 双指针
        int index, detect;
        index = 0; detect = 0;
        while(detect < nums.length){
            if(nums[detect] != val){
                nums[index ++] = nums[detect];
            }
            detect ++;
        }
        return index;
    }
}
```
