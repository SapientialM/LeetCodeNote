# [58. 区间和（第九期模拟笔试）](https://kamacoder.com/problempage.php?pid=1070)

我只能说，卡玛网的测试有点离谱，不是很懂。最后输入还需要加try-catch。

```java
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        // 长度 n
        int n = sc.nextInt();
        int pos = 1;
        int sums[] = new int[n+1];
        sums[0] = 0;
        // n 行输入数组
        for(int i = 1; i <= n; i ++){
            sums[i] = sums[i - 1] + sc.nextInt();
        }
        // 计算区间 a, b
        while(sc.hasNextLine()){
            try{
                int a = sc.nextInt();
                int b = sc.nextInt();
                System.out.println(sums[b+1] - sums[a]);
            }
            catch(Exception e){
                break;
            }
        }
    }
}

```

另外代码过程中做了一个大坑。不要这么用`nums[pos ++] = nums[pos -1] + sc.nextInt()`，因为`pos++`会在该次使用pos后提前结算 pos 的值。
