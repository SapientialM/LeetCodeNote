# [704. 二分查找](https://leetcode.cn/problems/binary-search/)

```java
class Solution {
    public int search(int[] nums, int target) {
        // 给的是升序数组
        int be = 0, en = nums.length - 1;
        while(be < en){
            // 这里会向左取整，所以 be 要 + 1
            int mid = (be + en) / 2;
            int val = nums[mid];
            if(val == target){
                return mid;
            }
            if(target < val){
                en = mid;
            }
            else{
                be = mid + 1;
            }
        }
        if(nums[be] != target){
            return -1;
        }
        return be;
    }
}
```

注意：向左取整，这道题要秒答
