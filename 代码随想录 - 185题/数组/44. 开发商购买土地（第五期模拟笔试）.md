# [44. 开发商购买土地（第五期模拟笔试）](https://kamacoder.com/problempage.php?pid=1044)

两个前缀和即可

```java
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        // n * m
        final int n = sc.nextInt();
        final int m = sc.nextInt();
        int matrix[][] = new int[n][m];
        // 维护两个前缀和，行和列
        int colSum[] = new int[m];
        int rowSum[] = new int[n];
        for(int i = 0; i < n; i ++){
            int sum = 0;
            for(int j = 0; j < m; j ++){
                matrix[i][j] = sc.nextInt();
                sum += matrix[i][j];
            }
            if(i == 0) rowSum[i] = sum;
            else rowSum[i] = rowSum[i-1] + sum;
        }
        for(int j = 0; j < m; j ++){
            int sum = 0;
            for(int i = 0; i < n; i ++){
                sum += matrix[i][j];
            }
            if(j == 0) colSum[j] = sum;
            else colSum[j] = colSum[j-1] + sum;
        }
        int minDiff = Integer.MAX_VALUE;
        // 判断
        for(int i = 0; i < n; i ++){
            minDiff = Math.min(minDiff, Math.abs(rowSum[n-1] - rowSum[i]*2));
        }
        for(int i = 0; i < m; i ++){
            minDiff = Math.min(minDiff, Math.abs(colSum[m-1] - colSum[i]*2));
        }
        System.out.println(minDiff);
    }
}

```
