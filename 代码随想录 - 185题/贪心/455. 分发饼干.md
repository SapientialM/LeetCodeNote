# [455. 分发饼干](https://leetcode.cn/problems/assign-cookies/)

贪心入门，

```java
class Solution {
    public int findContentChildren(int[] g, int[] s) {
        // 排序
        Arrays.sort(g);
        Arrays.sort(s);
        int res = 0;
        for(int i = 0; i < g.length; i ++){
            // 最小饼干满足最小weikou
            for(int j = 0; j < s.length; j ++){
                if(g[i] <= s[j]){
                    res ++;
                    s[j] = -1;
                    break;
                }
            }
        }
        return res;
    }
}
```
