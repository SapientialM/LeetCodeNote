# [135. 分发糖果](https://leetcode.cn/problems/candy/)

向左向右单独遍历，同时满足条件会很复杂，单独分别满足会简单，也就是拆分。

```java
class Solution {
    public int candy(int[] ratings) {
        int candys[] = new int[ratings.length];
        Arrays.setAll(candys, a->1);
        // 向右
        for(int i = 1; i < candys.length; i ++){
            if(ratings[i] > ratings[i-1]) candys[i] = candys[i-1] + 1;
        }
        for(int i = candys.length - 2; i >= 0; i --){
            // 因为有可能我已经比你大了，
            if(ratings[i] > ratings[i+1]) candys[i] = Math.max(candys[i], candys[i+1] + 1);
        }

        int res = 0;
        for(int i : candys){
            res += i;
        }
        return res;
    }
}
```
