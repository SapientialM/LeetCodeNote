# [860. 柠檬水找零](https://leetcode.cn/problems/lemonade-change/)

最简单的处理方式：
```java
class Solution {
    public boolean lemonadeChange(int[] bills) {
        int pay[] = new int[3];
        for(int i : bills){
            if(i == 5){
                pay[0] ++;
            }
            else if(i == 10){
                pay[1] ++;
                pay[0] --;
            }
            else{
                if(pay[1] > 0){
                    pay[1] --;
                    pay[0] --;
                }
                else{
                    pay[0] -= 3;
                }
            }
            for(int p : pay){
                if(p < 0) return false;
            }
        }
        return true;
    }
}
```
