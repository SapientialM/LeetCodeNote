# [503. 下一个更大元素 II](https://leetcode.cn/problems/next-greater-element-ii/)

表面上循环，实际上直接弄一个 2*n 长度的数组遍历即可

```java
class Solution {
    public int[] nextGreaterElements(int[] nums) {
        int n = nums.length;
        int[] res = new int[n];
        Arrays.fill(res, -1);
        Deque<Integer> stack = new ArrayDeque<>();
        // 循环数组，遍历两遍长度
        for (int i = 0; i < 2 * n; i++) {
            int actualIndex = i % n;  // 实际索引
            int currentNum = nums[actualIndex];
            // 单调栈逻辑
            while (!stack.isEmpty() && currentNum > nums[stack.peek()]) {
                int prevIndex = stack.pop();
                res[prevIndex] = currentNum;
            }
            // 只在第一轮遍历时入栈
            if (i < n) {
                stack.push(actualIndex);
            }
        }
        return res;
    }
}
```
