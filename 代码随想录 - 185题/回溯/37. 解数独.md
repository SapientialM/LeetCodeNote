# [37. 解数独](https://leetcode.cn/problems/sudoku-solver/)

 深度默认为9*9，二维遍历，铺开进行回溯。注意找到解法后需要返回，不用继续找了。因为解法只有一个。

```java
class Solution {
    public void solveSudoku(char[][] board) {
        // 暂时都用遍历
        backtracking(board, 0, 0);
    }
    private boolean backtracking(char[][] board, int row, int col){
        if(row == 9) return true;
        // 如果不为空，就去填下一个
        int nextRow = col == 8? row+1 : row;
        int nextCol = col == 8? 0 : col+1;
        if(board[row][col] != '.'){
            return backtracking(board, nextRow, nextCol);
        }
        // 做二维遍历，深度为 9*9
        for(int i = 1; i < 10; i ++){
            // 看看填什么
            char num = (char)('0' + i);
            if(check(board, num, row, col)){
                board[row][col] = num;
                if(backtracking(board, nextRow, nextCol) == true){
                    return true;
                }
                board[row][col] = '.';
            }
        }
        return false;
    }
    private boolean check(char[][] board, char num, int row, int col){
        // 行列check
        for(int i = 0; i < 9; i ++){
            if(board[row][i] == num) return false;
        }
        for(int i = 0; i < 9; i ++){
            if(board[i][col] == num) return false;
        }
        // 矩阵check
        int matrixRow = row/3;
        int matrixCol = col/3;
        for(int i = matrixRow*3; i < matrixRow*3 + 3; i ++){
            for(int j = matrixCol*3; j < matrixCol*3 + 3; j ++){
                if(board[i][j] == num) return false;
            }
        }
        return true;
    }
}

```
