# [90. 子集 II](https://leetcode.cn/problems/subsets-ii/)

HashSet + 排序，这个思路比较好想通，是通过排序后的toString来比较的。

```java
class Solution {
    List<Integer> path;
    List<List<Integer>> res;
    HashSet<String> set;
    public List<List<Integer>> subsetsWithDup(int[] nums) {
        res = new ArrayList<>();
        path = new ArrayList<>();
        set = new HashSet<>();
        Arrays.sort(nums);
        backtracking(nums, 0);
        return res;
    }
    private void backtracking(int[] nums, int be) {
        if (be == nums.length) {
            if(!set.contains(path.toString())){
                res.add(new ArrayList<>(path));
                set.add(path.toString());    
            }
            return;
        }        
        // 不选当前元素
        backtracking(nums, be + 1);
        // 选当前元素
        path.add(nums[be]);
        backtracking(nums, be + 1);
        path.remove(path.size() - 1);
    }

}
```

切割 + 剪枝，不用Set。

这个剪枝思路很简单，我们容易遇到的其实是 `2 2 2`这种，容易切割得到`2 2`和`2 2`，也就是下标12/13这种选择，但是不能排除 1/12/123这种选择，那就是要选就一直选，不选就全不选，也就是只能从1开头。

```java
class Solution {
    List<Integer> path;
    List<List<Integer>> res;
    
    public List<List<Integer>> subsetsWithDup(int[] nums) {
        res = new ArrayList<>();
        path = new ArrayList<>();
        Arrays.sort(nums);
        backtracking(nums, 0);
        return res;
    }
    
    private void backtracking(int[] nums, int index) {
        if (index == nums.length) {
            res.add(new ArrayList<>(path));
            return;
        }
        
        // 选择当前元素
        path.add(nums[index]);
        backtracking(nums, index + 1);
        path.remove(path.size() - 1);
        
        // 不选当前元素时，跳过所有相同的后续元素
        int next = index + 1;
        while (next < nums.length && nums[next] == nums[index]) {
            next++;
        }
        backtracking(nums, next);
    }
}

```
