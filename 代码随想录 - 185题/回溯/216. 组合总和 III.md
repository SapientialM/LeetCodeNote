# [216. 组合总和 III](https://leetcode.cn/problems/combination-sum-iii/)

开始的版本，就是最基础的回溯

```java
class Solution {
    public List<List<Integer>> combinationSum3(int k, int n) {
        // 表面上无序，实际上以有序来做
        List<Integer> path = new ArrayList<>();
        List<List<Integer>> res = new ArrayList<>();
        backtracking(path, res, 0, k, n);
        return res;
    }
    private void backtracking(List<Integer> path, List<List<Integer>> res, int s, int k, int sumN){
        if(path.size() == k){
            int sum = 0;
            for(int i : path){
                sum += i;
            }
            if(sum == sumN){
                res.add(new ArrayList<>(path));
            }
        }
        for(int i = s+1; i < 10; i ++){
            path.add(i);
            backtracking(path, res, i, k, sumN);
            path.remove(path.size() - 1);
        }
    }
}
```

很明显有很多可以剪枝的地方，所以进行剪枝，包括剩余的数量不够、剩余的和不够等。

```java
class Solution {
    public List<List<Integer>> combinationSum3(int k, int n) {
        List<List<Integer>> res = new ArrayList<>();
        backtrack(res, new ArrayList<>(), 1, k, n);
        return res;
    }
    
    private void backtrack(List<List<Integer>> res, List<Integer> path, int start, int k, int remaining) {
        if (path.size() == k && remaining == 0) {
            res.add(new ArrayList<>(path));
            return;
        }
        
        // 剪枝条件：
        // 1. 剩余数字不够填满k个：9 - i + 1 < k - path.size()
        // 2. 当前数字已经大于剩余值
        for (int i = start; i <= 9; i++) {
            if (i > remaining) break; // 剪枝：当前数字已经太大
            if (9 - i + 1 < k - path.size()) break; // 剪枝：剩余数字不够
            
            path.add(i);
            backtrack(res, path, i + 1, k, remaining - i);
            path.remove(path.size() - 1);
        }
    }
}

```
