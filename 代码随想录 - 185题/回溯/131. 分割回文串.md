# [131. 分割回文串](https://leetcode.cn/problems/palindrome-partitioning/)

```java
class Solution {
    List<List<String>> res;
    List<String> path;
    public List<List<String>> partition(String s) {
        // 切割问题其实类似组合问题
        res = new ArrayList<>();
        path = new ArrayList<>();
        backtracking(s);
        return res;
    }

    private void backtracking(String s){
        if(s.length() == 0){
            res.add(new ArrayList<>(path));
            return;
        }
        for(int i = 1; i <= s.length(); i ++){
            String tmpStr = s.substring(0, i);
            if(isPali(tmpStr)){
                path.add(tmpStr);
                backtracking(s.substring(i));
                path.remove(path.size() - 1);
            }
            else{
                continue;
            }
        }    
    }

    private boolean isPali(String s){
        int be = 0, en = s.length() - 1;
        while (be < en) {
            if(s.charAt(be) == s.charAt(en)){
                be ++;
                en --;
            }
            else{
                return false;
            }
        }
        return true;
    }
}
```
