# [78. 子集](https://leetcode.cn/problems/subsets/)

切割回溯思路：

```java
class Solution {
    List<Integer> path;
    List<List<Integer>> res;
    public List<List<Integer>> subsets(int[] nums) {
        // 非常简单的切割回溯
        res = new ArrayList<>();
        path = new ArrayList<>();
        backtracking(nums, 0);
        return res;
    }
    private void backtracking(int[] nums, int be){
        // 每个节点都可以是一个子集
        res.add(new ArrayList<>(path));
        if(be == nums.length){
            return;
        }
        // 只需要包含/不包含  
        for(int i = be; i < nums.length; i ++){
            path.add(nums[i]);  
            backtracking(nums, i+1);
            path.remove(path.size() - 1);
        }
        
    }
}
```

二叉树思路

```java
class Solution {
    List<Integer> path;
    List<List<Integer>> res;
    public List<List<Integer>> subsets(int[] nums) {
        res = new ArrayList<>();
        path = new ArrayList<>();
        backtracking(nums, 0);
        return res;
    }
    private void backtracking(int[] nums, int index) {
        if (index == nums.length) {
            res.add(new ArrayList<>(path));
            return;
        }
        
        // 不选当前元素
        backtracking(nums, index + 1);
        
        // 选当前元素
        path.add(nums[index]);
        backtracking(nums, index + 1);
        path.remove(path.size() - 1);
    }

}
```
