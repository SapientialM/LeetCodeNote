# [47. 全排列 II](https://leetcode.cn/problems/permutations-ii/)

最简单的思路就是 46.全排列 基础上加一个哈希即可解决

```java
class Solution {
    List<Integer> path; 
    List<List<Integer>> res;
    HashSet<String> sets;
    public List<List<Integer>> permuteUnique(int[] nums) {
        // 全排列，则固定长度，包括顺序，其实就是一个N叉树
        path = new ArrayList<>();
        res = new ArrayList<>();
        sets = new HashSet<>();
        int used[] = new int[nums.length];
        for(int i = 0; i < used.length; i ++) used[i] = 0;
        backtracking(nums, used);
        return res;
    }
    private void backtracking(int[] nums, int[] used){
        if(path.size() == nums.length){
            String tmp = path.toString();
            if(sets.contains(tmp)) return;
            sets.add(tmp);
            res.add(new ArrayList<>(path));
            return;
        }
        for(int i = 0; i < nums.length; i ++){
            if(used[i] == 1) continue;
            path.add(nums[i]);
            used[i] = 1;
            backtracking(nums, used);
            path.remove(path.size() - 1);
            used[i] = 0;
        }
    }
}
```

但是复杂度更高，可以通过剪枝解决这个问题，简称“动脑子”，其实这道题和 90. 子集 II 题目的想法是一样的`不选当前元素，则全不选。若选则全选`，也就保证了只存在例如`0 1 1 2`的序列里，`1 1`序列只被选择一次，而不会重复选择，因为只会存在`1 1`或`1 x`两种序列，不会正反互选。

```java
class Solution {
    List<Integer> path; 
    List<List<Integer>> res;
    public List<List<Integer>> permuteUnique(int[] nums) {
        // 全排列，则固定长度，包括顺序，其实就是一个N叉树
        path = new ArrayList<>();
        res = new ArrayList<>();
        int used[] = new int[nums.length];
        for(int i = 0; i < used.length; i ++) used[i] = 0;
        Arrays.sort(nums);
        backtracking(nums, used);
        return res;
    }
    private void backtracking(int[] nums, int[] used){
        if(path.size() == nums.length){
            res.add(new ArrayList<>(path));
            return;
        }
        for(int i = 0; i < nums.length; i ++){
            if(used[i] == 1) continue;
            // 这里做一次剪枝
            if (i > 0 && nums[i] == nums[i - 1] && used[i - 1] == 0) {
                continue;
            }
            path.add(nums[i]);
            used[i] = 1;
            backtracking(nums, used);
            path.remove(path.size() - 1);
            used[i] = 0;
        }
    }
}
```
