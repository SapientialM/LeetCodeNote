# [17. 电话号码的字母组合](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/)

构建一个号码表，然后回溯，String无法进行动态处理，所以用StringBuilder

```java
class Solution {
    private String[] mapping = {
        "", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz"
    };
    
    public List<String> letterCombinations(String digits) {
        List<String> res = new ArrayList<>();
        if (digits == null || digits.length() == 0) return res;
        // StringBuilder 更好用
        StringBuilder path = new StringBuilder();
        backtracking(0, digits, path, res);
        return res;
    }
    
    private void backtracking(int start, String digits, StringBuilder path, List<String> res) {
        if (start == digits.length()) {
            res.add(path.toString());
            return;
        }
        
        int digit = digits.charAt(start) - '0';
        String letters = mapping[digit];
        
        for (int i = 0; i < letters.length(); i++) {
            path.append(letters.charAt(i));
            backtracking(start + 1, digits, path, res);
            path.deleteCharAt(path.length() - 1);
        }
    }
}

```
