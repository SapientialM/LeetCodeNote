# [249. 移位字符串分组](https://leetcode.cn/problems/group-shifted-strings/)

```java
class Solution {
    public List<List<String>> groupStrings(String[] strings) {
        // 保存差值序列的字符串
        // a b c -> 1 1 1
        HashMap<String, List<String>> map = new HashMap<>();
        for(String str : strings){
            ArrayList<Integer> tmpList = new ArrayList<>();
            Character pre = str.charAt(0);
            for(Character c : str.toCharArray()){
                int tmpNum = (c - pre + 26)% 26;
                tmpList.add(tmpNum);
                pre = c;
            }
            String keyStr = tmpList.toString();
            if(map.containsKey(keyStr)){
                List<String> mapList = map.get(keyStr);
                mapList.add(str);
            }
            else{
                map.put(keyStr, new ArrayList<>(Arrays.asList(str)));
            }
        }
        List<List<String>> resGroup = new ArrayList<>();
        for(String Key : map.keySet()){
            resGroup.add(map.get(Key));
        }
        return resGroup;
    }
}
```

注意：

`new ArrayList<>(Arrays.asList(str))`可变，而`Arrays.asList(str)`不可变，且`mapList`是引用，不需要`replace`
