# [2257. 统计网格图中没有被保卫的格子数](https://leetcode.cn/problems/count-unguarded-cells-in-the-grid/)

和BFS差不多：

1. **清晰的标记系统**：
   - `0`: 未被守卫的空地
   - `1`: 被守卫的空地
   - `2`: 墙
   - `3`: 守卫
2. **按方向扫描**：
   - 每行从左到右和从右到左扫描
   - 每列从上到下和从下到上扫描
3. **正确的阻挡逻辑**：
   - 遇到墙时重置守卫状态
   - 遇到守卫时开始守卫
   - 只在空地且被守卫时才标记为被守卫
4. **时间复杂度**：O(m×n)

```java
class Solution {
    public int countUnguarded(int m, int n, int[][] guards, int[][] walls) {
        // 0: 未被守卫的空地, 1: 被守卫, 2: 墙, 3: 守卫
        int[][] grid = new int[m][n];
        
        // 标记墙
        for (int[] wall : walls) {
            grid[wall[0]][wall[1]] = 2;
        }
        
        // 标记守卫
        for (int[] guard : guards) {
            grid[guard[0]][guard[1]] = 3;
        }
        
        // 按行扫描
        for (int i = 0; i < m; i++) {
            // 从左到右扫描
            boolean guarded = false;
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == 2) { // 遇到墙，重置守卫状态
                    guarded = false;
                } else if (grid[i][j] == 3) { // 遇到守卫，开始守卫
                    guarded = true;
                } else if (guarded && grid[i][j] == 0) { // 被守卫的空地
                    grid[i][j] = 1;
                }
            }
            
            // 从右到左扫描
            guarded = false;
            for (int j = n - 1; j >= 0; j--) {
                if (grid[i][j] == 2) { // 遇到墙，重置守卫状态
                    guarded = false;
                } else if (grid[i][j] == 3) { // 遇到守卫，开始守卫
                    guarded = true;
                } else if (guarded && grid[i][j] == 0) { // 被守卫的空地
                    grid[i][j] = 1;
                }
            }
        }
        
        // 按列扫描
        for (int j = 0; j < n; j++) {
            // 从上到下扫描
            boolean guarded = false;
            for (int i = 0; i < m; i++) {
                if (grid[i][j] == 2) { // 遇到墙，重置守卫状态
                    guarded = false;
                } else if (grid[i][j] == 3) { // 遇到守卫，开始守卫
                    guarded = true;
                } else if (guarded && grid[i][j] == 0) { // 被守卫的空地
                    grid[i][j] = 1;
                }
            }
            
            // 从下到上扫描
            guarded = false;
            for (int i = m - 1; i >= 0; i--) {
                if (grid[i][j] == 2) { // 遇到墙，重置守卫状态
                    guarded = false;
                } else if (grid[i][j] == 3) { // 遇到守卫，开始守卫
                    guarded = true;
                } else if (guarded && grid[i][j] == 0) { // 被守卫的空地
                    grid[i][j] = 1;
                }
            }
        }
        
        // 统计未被守卫的格子
        int count = 0;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == 0) {
                    count++;
                }
            }
        }
        
        return count;
    }
}
```
