# [232. 用栈实现队列](https://leetcode.cn/problems/implement-queue-using-stacks/)

```java
class MyQueue {
    private Stack<Integer> stPop, stPush;
    public MyQueue() {
        stPop = new Stack<>();
        stPush = new Stack<>();
    }
    
    public void push(int x) {
        stPush.push(x);
    }
    
    public int pop() {
        if(stPop.empty()){
            while(!stPush.empty()){
                stPop.push(stPush.pop());
            }
        }
        return stPop.pop();
    }
    
    public int peek() {
        if(stPop.empty()){
            while(!stPush.empty()){
                stPop.push(stPush.pop());
            }
        }
        return stPop.peek();
    }
    
    public boolean empty() {
        if(stPush.size() + stPop.size() == 0){
            return true;
        }
        else{
            return false;
        }
    }
}

/**
 * Your MyQueue object will be instantiated and called as such:
 * MyQueue obj = new MyQueue();
 * obj.push(x);
 * int param_2 = obj.pop();
 * int param_3 = obj.peek();
 * boolean param_4 = obj.empty();
 */
```
