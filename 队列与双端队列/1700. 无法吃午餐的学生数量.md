# [1700. 无法吃午餐的学生数量](https://leetcode.cn/problems/number-of-students-unable-to-eat-lunch/)

```java
class Solution {
    public int countStudents(int[] students, int[] sandwiches) {
        Queue<Integer> stuQeque = new ArrayDeque<>();
        for(int s : students){
            stuQeque.offer(s);
        }
        int pos = 0;
        int disLen = 0;
        while(disLen < stuQeque.size() && !stuQeque.isEmpty()){
            int s = stuQeque.poll();
            if(sandwiches[pos] == s){
                // 喜欢，拿走且离开
                pos ++;
                disLen = 0;
            }
            else{
                disLen ++;
                // 不喜欢，回到尾部
                stuQeque.offer(s);
            }
        }
        return stuQeque.size();
    }
}
```
