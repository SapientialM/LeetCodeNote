# [950. 按递增顺序显示卡牌](https://leetcode.cn/problems/reveal-cards-in-increasing-order/)

```java
class Solution {
    public int[] deckRevealedIncreasing(int[] deck) {
        Arrays.sort(deck);
        // 1 2 3 4 5 6 7 8
        // 逆向模拟
        Stack<Integer> st = new Stack<>();
        for(int val : deck){
            st.push(val);
        }
        Deque<Integer> res = new ArrayDeque<>();
        while(!st.isEmpty()){
            // 从牌堆底部放一张牌上来
            if(!res.isEmpty())
                res.offerLast(res.pollFirst());
            // 从结果顶部一张牌放回
            res.offer(st.pop());

        }
        int[] resDeck = new int[res.size()];
        int i = res.size()-1;
        for(int val : res){
            resDeck[i --] = val;
        }
        return resDeck;
    
    }
}
```

主要是要考虑复现顺序。
