# [121. 买卖股票的最佳时机](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/)

```java
class Solution {
    public int maxProfit(int[] prices) {
        // 维护按顺序的最高值与最低值的差值
        Stack<Integer> st = new Stack<>();
        int maxProfit = 0;
        for(int i = 0; i < prices.length; i ++){
            int value = prices[i];
            // 如果有更低的出现了就会覆盖全部，所以队尾是最大值，队头是最小值，逆序队列
            while(!st.empty() && value < st.peek()){
                st.pop();
            }
            st.push(value);
            // cal
            maxProfit = Math.max(st.getLast() - st.getFirst(), maxProfit);
        }
        return maxProfit;
    }
}
```
