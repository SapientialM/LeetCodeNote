# [346. 数据流中的移动平均值](https://leetcode.cn/problems/moving-average-from-data-stream/)

```java
class MovingAverage {
    Queue<Integer> innerData;
    int sumData;
    int windowSize;
    public MovingAverage(int size) {
        sumData = 0;
        windowSize = size;
        innerData = new ArrayDeque<>();
    }
    
    public double next(int val) {
        if(innerData.size() >= windowSize){
            sumData -= innerData.poll();
        }
        innerData.offer(val);
        sumData += val;
        double sum = sumData;
        double window = innerData.size();

        return sum/window;
    }
}

/**
 * Your MovingAverage object will be instantiated and called as such:
 * MovingAverage obj = new MovingAverage(size);
 * double param_1 = obj.next(val);
 */
```
